# Trippy

Trippy is a mobile-first travel planning app for keeping every part of a trip in one place: the itinerary, bookings, map, shared plans, and AI-assisted planning notes.

The app is designed around day-by-day trip planning. Travellers can create trips, organize stops across each day, add notes and timing details, browse destination suggestions, and view pinned stops on a map. It also includes a logistics hub for flights, trains, hotels, and other bookings, so confirmations and travel details stay close to the actual plan.

Trippy supports collaborative planning through shared trip access and public itinerary views. An in-app co-pilot can answer trip questions and suggest itinerary changes, with previews before updates are applied.

## Current Features

- Trip dashboard with active, upcoming, and past journeys
- Day-by-day itinerary timeline
- Editable stops with notes, timing, duration, cost, and location details
- Destination discovery by interest, with category tabs and surprise picks
- Map view for pinned itinerary stops
- Booking manager for flights, trains, hotels, and other reservations
- Share links and collaborator controls
- Public shared itinerary pages
- AI co-pilot for planning help and proposed itinerary edits
- Mobile-first interface with PWA-friendly behavior

Trippy is still in development, with the focus on making travel planning feel organized, collaborative, and pleasant to use on the go.
